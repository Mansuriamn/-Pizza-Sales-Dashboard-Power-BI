# 📌 Project Overview

Understanding sales data is essential for making informed business decisions. This dashboard analyzes pizza sales data and presents key business metrics through interactive visualizations.

The objective of this project was to transform raw transactional data into a meaningful dashboard that helps identify sales trends, customer behavior, product performance, and profitability.

This project is my second Power BI dashboard, built to strengthen my understanding of Business Intelligence, dashboard design, and data storytelling.

### 📈 Business KPIs

# 📚 What I Learned

This project helped me improve my understanding of designing dashboards that communicate business insights rather than simply displaying charts.

Some of my key learnings include:

- Building interactive dashboards using slicers
- Selecting the most appropriate visualizations for different business questions
- Designing cleaner and more user-friendly layouts
- Creating meaningful KPI cards
- Organizing reports for better readability
- Converting raw sales data into actionable business insights

One important lesson from this project is:

> **Every dashboard tells a story. The goal isn't just to build charts—it's to help users understand what the data is saying.**
